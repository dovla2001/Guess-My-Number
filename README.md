# 🎯 Guess My Number

Simple number guessing game built with JavaScript.

The player has to guess a randomly generated number between 1 and 20.
Score decreases with each wrong guess.

---

## 🚀 Features

- Random number generation
- Score tracking
- Highscore system
- Dynamic DOM manipulation
- Game reset functionality

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
